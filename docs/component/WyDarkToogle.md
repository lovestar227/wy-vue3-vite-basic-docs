# WyDarkToogle 明亮/暗黑模式开关

- 明亮/暗黑模式开关
- 依赖 @vueuse/core 明暗模式切换功能
- 依赖 WySwitch 开关组件

## WyDarkToogle Events

| 事件名        | 说明           | 类型  | 定义 |
| ------------- | ------------- | ----- | ----- |
| change | 开关状态切换事件 | Function  | ( value: boolean ) => void |

## 使用方式

- 基础使用

    ```html
    <!-- 切换明暗模式 -->
    <WyDarkToogle></WyDarkToogle>
    ```
